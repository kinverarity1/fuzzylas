Warning! This web app is in early development. There are bugs and the data is old and rough.

What is fuzzyLAS?

What? This web application does a fuzzy lookup of wireline log mnemonics. Given any string, it returns the best guess from its database.

Whence? The database was compiled and curated from the resources on the SPWLA's website.

How? There are two ways to use the app: go to the home page and use the form; or use the web API.

Why? The app is an experiment to see if we could improve on existing curve search, especially providing a machine-readable output, and to help us learn to program.
